# Contributing to DarkLead

This project was autonomously generated by Claude Sonnet 4.6.

## Development Setup

```bash
# Backend
cd backend
pip install -r requirements.txt --break-system-packages
cp ../.env.example .env  # Edit ANTHROPIC_API_KEY or USE_LOCAL_LLM=true
python seed_demo.py
uvicorn app.main:app --port 7860 --reload

# Frontend
cd frontend
npm install --legacy-peer-deps
npm run dev  # http://localhost:5173
```

## Running with Ollama (local, free)

```bash
ollama pull qwen2.5-coder:14b
USE_LOCAL_LLM=true OLLAMA_MODEL=qwen2.5-coder:14b uvicorn app.main:app --port 7860
```
