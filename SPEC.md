## Scan Pipeline (13 Steps)

1. Clone repo (gitpython, size-limited)
2. Detect language (file extension frequency)
3. Count LOC per module
4. Compute git churn (commit frequency per file)
5. Run 19 scanners in parallel (asyncio.gather)
6. LLM triage: dedup, re-classify severity, add OWASP/CWE
7. LLM fix generation: unified diff per finding
8. LLM misconfig remediation: corrected IaC snippets
9. Module aggregation: debt score per file
10. Compliance mapping: OWASP Top10 + NIST CSF
11. Dependency update check: MANDATORY/SUGGESTED/OPTIONAL
12. Persist all results to SQLite
13. Emit WebSocket complete event
