## What is DarkLead?

DarkLead is a production-grade **Static Application Security Testing (SAST)** platform that combines:

- **19 scanning tools** (bandit, ruff, lizard, semgrep, gitleaks, trivy, pip-audit, and more)
- **Local LLM analysis** via Ollama (`qwen2.5-coder:14b`) — no data leaves your machine
- **Anthropic Claude** as optional cloud backend
- A **professional FortiAnalyzer-themed dashboard** with 20+ panels

Submit any GitHub repository URL → DarkLead clones it, runs all scanners in parallel, has the LLM triage and explain every finding, generates AI-suggested fixes, and delivers a prioritized remediation report with debt score + letter grade.

## 30-Second Demo

```bash
# Start the stack
cd darklead
make install
make backend  # FastAPI on :7860

# In another terminal
make frontend  # Vite dev on :5173

# Scan a repo
curl -X POST http://localhost:7860/api/scan \
  -H "Content-Type: application/json" \
  -d '{"repo_url": "https://github.com/owner/repo"}'
```

Or open `http://localhost:7860/?demo=demo-darklead-19c2af76` for the seeded demo.

## Key Features

| Feature | Detail |
|---------|--------|
| **Ensemble SAST** | 19 tools run in parallel, results merged and deduplicated |
| **LLM Triage** | Every finding gets CWE, OWASP category, severity explanation |
| **AI Fix Suggestions** | Diff-ready code fixes generated by qwen2.5-coder:14b |
| **Debt Score** | 0–100 composite score → A/B/C/D/F grade |
| **Real-time Progress** | WebSocket push for live 13-step pipeline status |
| **SARIF Export** | GitHub Code Scanning compatible output |
| **CycloneDX SBOM** | Software Bill of Materials for dependency tracking |
| **CVE Advisory Feed** | NVD + OSV.dev polling with SSE push to frontend |
| **AI Chat** | Ask the LLM questions about any scan result |
| **Secret Scanner** | 8 credential pattern types (API keys, AWS, GitHub PAT…) |
| **SQL Injection Detector** | 5 injection patterns as pre-LLM gate |
| **Config Auditor** | YAML + .env misconfiguration detection |
| **GitHub Webhook** | Auto-trigger scan on every push |
